# Competitive Feature Benchmark (경쟁 기능 벤치마크)

경쟁 제품이 유사한 기능을 UX/인터랙션 수준에서
어떻게 구현하는지 조사하고 비교합니다.

이 스킬은:

- 동일 도메인에서 3~5개 관련 경쟁사를 식별
- 경쟁사별 정보 구조, 내비게이션, 확장성, 파워유저 지원, 엣지 케이스 처리를 분석
- 구조화된 비교표를 생성
- 공통 패턴, 이상치, 안티패턴을 종합
- 전략적 차별화 권고안을 제공

---

## 왜 필요한가

기능 설계 결정은 의견이 아닌 근거에 기반해야 합니다.
경쟁사가 같은 문제를 어떻게 해결하는지 이해하면
검증된 패턴, 피해야 할 함정, 차별화 기회를 발견할 수 있습니다.
단순 모방이 아닌 전략적 분석입니다.

---

## 분석 차원

- **정보 구조** — 평면 vs 계층, 그룹핑, 컨텍스트 보존
- **내비게이션 & 인터랙션** — 클릭 깊이, 점진적 공개, 컨트롤 유형
- **확장성 처리** — 대규모 데이터셋, 검색/필터/정렬, 지연 로딩
- **파워유저 지원** — 일괄 작업, 키보드 단축키, 커스터마이징
- **엣지 케이스 처리** — 빈 상태, 권한, 긴 레이블, 깊은 중첩
