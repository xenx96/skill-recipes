# Security Policy

## Scope

This repository contains LLM agent skill definitions (Markdown files).
It does not include executable server code or services.

However, skills may reference external APIs (e.g., Notion MCP) and
include shell command examples. Security concerns related to these
references are in scope.

## Reporting a Vulnerability

If you discover a security issue (e.g., a skill that could lead to
unintended data exposure, credential leakage in examples, or unsafe
shell command patterns), please report it responsibly:

1. **Do NOT open a public issue.**
2. Email the maintainer or use [GitHub Security Advisories](https://github.com/comsky/remy-skill-recipes/security/advisories/new) to report privately.
3. Include a description of the issue, affected files, and potential impact.

## Response

- Acknowledgment within 48 hours
- Fix or mitigation within 7 days for confirmed issues
- Credit given to reporters (unless anonymity is requested)
