# Remy Skill Recipes (한국어 안내)

이 레포는 LLM을 활용한 엔지니어링 작업을 구조화하기 위한
“스킬 레시피” 모음입니다.

단순 프롬프트 모음이 아니라,
재사용 가능하고 실패 확률을 줄이는 구조화된 워크플로우를 제공합니다.

---

## 목적

- 디버깅
- 설계 문서 작성
- UI/UX 리뷰
- 경쟁 제품 분석
- 코드 변경 영향 분석

등의 반복 업무를 표준화하기 위함입니다.

---

## 사용 방법

1. `skills/` 폴더에서 필요한 스킬을 선택합니다.
2. **Inputs Required** 섹션을 반드시 확인합니다.
3. 필요한 컨텍스트를 충분히 제공합니다.
4. 출력 결과를 체크리스트 기준으로 검증합니다.

입력 정보가 부족하면 결과 품질이 급격히 낮아집니다.

---

## 스킬 설계 원칙

각 스킬은 다음을 반드시 포함합니다:

- Purpose (목적)
- When to Use / When NOT to Use
- Inputs Required (필수 입력 정보)
- Output Format (출력 형식 정의)
- Guardrails (환각 방지 장치)
- Failure Patterns (잘못된 결과 유형)
- Examples (현실적인 예시 최소 2개)

---

## 이 레포의 방향성

- 모델 종속성 최소화
- 구조 중심 설계
- 실무 재사용 가능성 최우선
- 과도한 프롬프트 미화 지양

LLM은 구조 없이 쓰면 불안정합니다.
이 레포는 그 구조를 제공합니다.
